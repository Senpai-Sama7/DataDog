# Decision-first Deck Outline (6–8 slides)

1. **Decision & Audience** — State the decision, owner, and deadline.
2. **Baseline & Target** — Show current metric vs target; define cohort/time window.
3. **Smallest Analysis** — The minimum evidence that could change the decision.
4. **Finding** — One chart with labeled axes and an annotated threshold.
5. **Risk & Assumption** — One explicit risk and how you would falsify it.
6. **Recommendation** — One sentence action, cost, expected impact, next step.
7–8. **Appendix (optional)** — Data source, glossary, caveats.
